# Tic Tac Toe Game

## Overview

This is a simple Tic Tac Toe game built using basic web technologies. The project demonstrates fundamental programming concepts such as event handling, game logic, and user interaction.

## Features

* Two player mode
* Interactive game board
* Win and draw detection
* Restart game functionality
* Simple and clean user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run the Project

1. Clone the repository.
2. Open the project folder.
3. Run the `index.html` file in your browser.

## Project Purpose

This project was created as a practice project to improve my understanding of JavaScript and front-end development.
